HOW TO USE GIT BASH AND HOW UTILIZE GIT-BASH FOR UPLOADING TO GITHUB.

GIT BASH is a possible method that can be used for uploading files and I'm gonna explain how I managed to use it step by step and I will show with an example how it works when I need to upload new files. First at all I needed to understand how to connect an existing repository using Git Bash on my computer.

Following the steps that explains the process:

• Step 1
➡ Install git bash

• Step 2
➡ Go to website folder on my computer

• Step 3
➡ Shift-right-click empty space in the folder
➡ Click "Open Git Bash here"

• Step 4
➡ Write the following:
- git init -b main
- git add .
- git commit -m "first commit"
- git config --global user.email "myemail@email.com"
- git remote add origin [repo url]
- git push --force origin main

Note:
I used --force because git bash thought the local branch was older than the remote branch. Normally do not use --force as it overrides all changes made on github by potential collaborators.
