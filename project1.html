Project 1

In this project I will talk about how I created and published this website step by step and my decision making and challenges during the project for online portfolio and documentation.

Project description

I have been using https://html5up.net/ to choose the template that I found the best option to describe my work during this course. I chose the template called “Story” because I liked how the material is organised and the design and graphics are very well presented in a way that attracts the reader.

The design shows the information with a practical approach and I chose colours that make an impact while reading.

A challenge I encounter while working on my website has been to create a subpage from the presentation of the first project, a subpage to which get access from pushing the bottom “Learn more” under each main chapter title. This challenge has been overcome understanding what to write in my code on brackets and how to do it.

I download the software https://brackets.io/ for the code editing environment of my website and selected the option Live preview to be able to see the changes and updating of my code right away on my website. I also downloaded https://git-scm.com/ but didn't use it because with a Live preview I don't need to command my computer to upload the code to the website repository project I created in https://github.com/. To create a repository project in Github I created an account which is the centre to which everything connects. So my website has a link to the Github repository.

In my website I will talk about myself with my resume I uploaded to the website and will present the project of this course in computer manufacturing. I think that many different topics will be covered and that it will be enjoyable to introduce them through this website. In my final project I would like to use the knowledge gained during the course and talk about manufacturing, design and materials choices that are fundamental basics of digital production technology and computer- aided production.
